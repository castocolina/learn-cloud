Why Python for Cloud?

What are the three main reasons Python is a strong choice for cloud-native development?

Answer

  1. Rich Ecosystem: Extensive libraries and frameworks (FastAPI, Boto3).
  2. Simplicity & Readability: Clean syntax leads to maintainable code.
  3. High Developer Productivity: Faster development cycles.

Type Hints

What is the primary purpose of type hints in Python? Do they enforce types at runtime?

Answer

Their primary purpose is to enable static analysis by tools like MyPy to catch type-related errors before the code is run. They do not enforce types at runtime; the Python interpreter ignores them.

Pydantic

What key capability does Pydantic add on top of Python's standard type hints?

Answer

Pydantic provides runtime data validation and serialization. It uses type hints to parse, validate, and transform data as the program is running, which is crucial for validating incoming data in APIs.

Data Structures

Which built-in Python data structure is most commonly used to represent a JSON object received in an API request?

Answer

The Dictionary (`dict`). Its key-value pair structure maps directly to the structure of a JSON object, making it the natural choice for serialization and deserialization.

Pydantic Special Types

What is the benefit of using special types like `EmailStr` or `HttpUrl` in a Pydantic model?

Answer

They provide built-in, format-specific validation. Pydantic will automatically check if a string is a valid email address or a well-formed URL, saving you the effort of writing complex validation logic yourself.
