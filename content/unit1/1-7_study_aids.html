1.7 Study Aids: Advanced Backend Topics

Use these flashcards to review the key concepts of advanced communication patterns in cloud-native backends.

Event-Driven Architecture (EDA)

What is the primary benefit of adopting an Event-Driven Architecture?

Answer

The primary benefit is decoupling services. Services communicate asynchronously via events, making them more independent, scalable, and resilient, as they don't need direct knowledge of each other.

Message Queues (RabbitMQ)

What is the primary role of a message queue like RabbitMQ in an event-driven system?

Answer

Message queues act as intermediaries, storing messages temporarily between producers and consumers. They enable asynchronous communication, buffer messages during traffic spikes, and facilitate reliable delivery and task offloading.

Kafka vs. RabbitMQ

What is the key difference in the primary use cases for Kafka compared to RabbitMQ?

Answer

RabbitMQ is generally used for traditional message queuing, task distribution, and point-to-point communication. Kafka is a distributed streaming platform designed for high-throughput, fault-tolerant event streams, real-time analytics, and event sourcing.

gRPC Advantages

List two main advantages of gRPC over REST for internal microservice communication.

Answer

1. Performance: Uses HTTP/2 and efficient binary serialization (Protobuf).
2. Strongly Typed Contracts: Enforces strict type checking and enables code generation across languages, reducing integration errors.

Protocol Buffers (Protobuf)

What is the role of Protocol Buffers in gRPC communication?

Answer

Protobuf is a language-neutral, platform-neutral mechanism for serializing structured data. In gRPC, it defines the service interfaces and message structures, ensuring efficient, strongly-typed data exchange between client and server.
