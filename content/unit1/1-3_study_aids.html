Flashcards: Code Quality and Standards

Question

What is PEP 8?

Answer

PEP 8 is Python's official style guide, providing conventions for writing readable and consistent Python code.

Question

What is the primary purpose of Black?

Answer

Black is an opinionated code formatter that automatically enforces a consistent code style, eliminating debates over formatting in code reviews.

Question

How does Ruff differ from Black?

Answer

Black is a code formatter (re-arranges code for style), while Ruff is a linter (analyzes code for potential errors, bugs, and stylistic issues).

Question

What are pre-commit hooks used for?

Answer

Pre-commit hooks are automated scripts that run before a commit is created to enforce code quality standards and ensure consistency in the codebase.

Question

Name two key areas covered by PEP 8.

Answer

Naming conventions (e.g., snake_case, PascalCase) and code layout (e.g., 4 spaces for indentation, line length limits).
