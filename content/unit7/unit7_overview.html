What You'll Learn

This unit explores serverless computing on AWS, covering Lambda functions, API Gateway, and event-driven architectures. Learn to build scalable, cost-effective applications using AWS serverless services.

Learning Objectives

  • Master AWS Lambda functions and serverless compute patterns
  • Build APIs using Amazon API Gateway
  • Implement event-driven architectures with AWS services
  • Use AWS SAM and Serverless Framework for deployment
  • Integrate with AWS data services (DynamoDB, S3, RDS)
  • Implement serverless monitoring and observability
  • Apply serverless security best practices
  • Design cost-effective serverless solutions
